Nox

Nox is a beautiful, easy to use and lightweight front-end framework built for the modern web.

Get Started

Built with Sass

Nox is built on top of the world's most powerful CSS preprocessor, SCSS. Made for customization, with tons of mixins so you can customize it to your heart's delight.

Flexbox

Nox uses the Flexbox grid from Bootstrap to provide a seamless 12-column grid.

Desktop First

In a world full of mobile-first frameworks, this one is built for desktop use first and foremost.

Fully Responsive

Nox is built for all platforms in mind, with compatability with both desktop and mobile experiences.

Quick Start

It's extremely easy to get started, simply include the CSS and JS and go.

Fast Load Times

All of our production files are minified to optimize load times. The Nox CSS sheet is only 25.9kb.
