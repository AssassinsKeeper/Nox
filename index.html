Nox

Nox is a beautiful, easy to use and lightweight front-end framework built for the modern web.

Get Started

Built with Sass

Nox is built on top of the world's most powerful CSS preprocessor, SCSS. Built with mixins, so it's easy to customize to your heart's delight.

Flexbox

Nox uses the Flexbox Grid from Bootstrap v4 to provide a seamless 12-column grid.

Desktop First

In a world full of mobile-first frameworks, this one is built for desktops first.

Fully Responsive

Nox is built for all platforms in mind, with compatibility with both desktop and mobile experiences

Quick Start

We have our own CDNs available so you can get started on your project as quickly as you can, or download and compile from source.

Fast Load Times

All of our production files are minified to optimize load times.
